需要切换到pulsar的环境，目前使用的是4.2版本

转换unet：
```
pulsar2 build --input out_onnx/unet_sim_cut.onnx --config config_unet_u16.json --output_dir output_unet_ax --output_name unet.axmodel
```
很慢，耗时可能长达一个小时，我花了68分钟

转换vae
```
pulsar2 build --input out_onnx/vae_decoder_sim.onnx --config config_vae_decoder_u16.json --output_dir output_vae_ax --output_name vae.axmodel
```



然后textencoder的ax模型可以在这里获取：`https://huggingface.co/yunyu1258/SD1.5-AX8850-xxmix9realistic/tree/main`
该仓库下还有前面过程中转换好的模型以及加载ax模型推理的python脚本，可以在板子上实测

整体耗时有：文本嵌入：0.7s+20步扩散：20x0.4s=8s+decoder：0.7s，合计约9.5秒
也可以自行调整部署为10步，实测不会画质影响不会太大
