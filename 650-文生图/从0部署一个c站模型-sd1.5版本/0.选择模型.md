该教程下的所有代码均位于：`https://github.com/yuyun2000/sd15-to-ax8850-deploy`
作者使用的开发板为 M5Stack-LLM 8850加速卡，该加速卡可插在树莓派或者其他开发板的m2接口，增加设备的ai运算能力

登录c站`https://civitai.com`，选择sd1.5的权重，随机挑一个模型
![](../../file/Pasted%20image%2020251126113036.png)


比如下面这个模型：
```
https://civitai.com/models/47274/xxmix9realistic
```
![](../../file/Pasted%20image%2020251126113503.png)
需要满足：
1、base model是sd1.5
2、模型类型是完整的checkpoint （lora理论上也可以，之后教程再出）
3、大小最好是1.99g 绝大多数也都是这样的大小，大一点问题也不大，可以自行尝试


没问题后下载得到一个 safetensor 权重
